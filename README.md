### Recommended Learning Approach

1. **Sequential Learning**: Follow the lectures in order (Lecture 1 â†’ Lecture 25)
2. **Theory First**: Read through the theoretical content in each lecture
3. **Practice**: Complete the practical exercises (named as `l[number]p[number].py`)

Files are named using the pattern: `l[lecture_number]p[practical_number]`
